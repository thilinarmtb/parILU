struct parilu_t *parilu_setup(const uint n, const slong *const vertex, const uint nnz, const uint *const row, const uint *const col, const double *const val, const struct parilu_opts_t *const options, const MPI_Comm comm, buffer *const bfr)

Setup parilu. Returns a pointer to a newly allocated struct parilu_t.

Parameters:

  • n – Number of local dofs in vertex array.

  • vertex – Array of local dofs with global numbering.

  • nnz – Number of nonzeros in the matrix.

  • row – Row indices of the matrix which points to a global dof in vertex array.

  • col – Column indices of the matrix which points to a global dof in vertex array.

  • val – Values of the matrix.

  • options – Pointer to the struct parilu_opts_t which contains the options.

  • comm – MPI communicator.

  • bfr – Pointer to the buffer struct used for work arrays.
